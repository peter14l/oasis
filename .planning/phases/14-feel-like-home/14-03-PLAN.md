# 14-03: Digital Garden

**Feature:** Digital Garden - Plant Thoughts  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

A private space where users can "plant" thoughts/seeds that grow over time into insights. Friends see only "sprouts", user sees full garden. Creates a personal growing space like tending a garden.

---

## PLAN

### Database
```sql
CREATE TABLE garden_plots (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  seed_text TEXT NOT NULL,  -- the original thought
  grown_insight TEXT,  -- evolved thought
  stage VARCHAR(20) DEFAULT 'seed',  -- seed, sprout, bloom
  days_to_grow INTEGER DEFAULT 30,
  planted_at TIMESTAMPTZ DEFAULT NOW(),
  last_tended_at TIMESTAMPTZ DEFAULT NOW()
);
```

### Step 1: UI - Garden View
1. Create GardenScreen in `lib/features/wellness/`
2. Grid of plots (circular, plant-like)
3. Tap to add seed

### Step 2: Growth Logic
1. Scheduler: after X days, seed → sprout
2. AI summary (optional): generate insight
3. User can "tend" to accelerate

### Step 3: Privacy
1. Complete privacy for user
2. Garden summary shows "🌱 Active plots: X" to friends
3. Full garden visible only to user
