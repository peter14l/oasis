# 14-09: Fortress Mode

**Feature:** Fortress Mode - One-Tap Lock  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

One-tap to lock the app with a custom "away" message. Friends see "In my fortress" instead of just "offline". Like hanging a "Do not disturb" sign - but warm.

---

## PLAN

### Step 1: Lock Mechanism
1. Same PIN/password as app lock
2. Long-press lock icon OR triple-tap
3. Instant activate

### Step 2: Away Messages
Predefined + custom:
- 🏰 "In my fortress" - Default
- 📵 "Digital detox" - No phones allowed
- 🔋 "Recharging"
- 🎯 "In the zone"
- Custom message

### Database
```sql
ALTER TABLE profiles ADD COLUMN fortress_mode BOOLEAN DEFAULT FALSE;
ALTER TABLE profiles ADD COLUMN fortress_message VARCHAR(200);
ALTER TABLE profiles ADD COLUMN fortress_until TIMESTAMPTZ;
```

### Step 3: Behavior
1. App locks immediately
2. Cannot access without PIN
3. On lock screen: "Away message" shown
4. Auto-disable if time set

### Step 4: UI
1. Lock icon in header (long-press to lock)
2. Lock screen shows away message
3. "Welcome back" on unlock

### Step 5: Notification to Friends
1. Last active shows: "In my fortress 🏰"
2. Not "last seen" - more comforting
3. DMs queue for later
