# 14-06: Welcome Wagon

**Feature:** Welcome Wagon - New Connection Welcome  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Customizable welcome messages for new connections. Replace cold "You have a new follower" with warm, privacy-first welcome that's actually helpful.

---

## PLAN

### Step 1: Welcome Templates
Default templates:
1. "Welcome to Oasis! 🎉 Your privacy matters - check Settings → Privacy to lock things down."
2. "Hey! Feel free to browse, but remember - what happens here stays here. ✨"
3. "Welcome! Questions? Check out the guide in Settings → Help."

### Step 2: Customization
1. User can create custom welcome
2. Preview before sending
3. Add privacy tips automatically (optional)

### Step 3: Trigger
1. When user A follows user B → auto-send welcome
2. When user joins a Circle → Circle welcome
3. First DM option

### Database
```sql
CREATE TABLE welcome_templates (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  template_text TEXT NOT NULL,
  is_default BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### Step 4: UI
1. Settings → Welcome Wagon section
2. Preview in settings
3. Enable/disable toggle
