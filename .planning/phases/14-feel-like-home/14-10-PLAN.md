# 14-10: Trust Circle

**Feature:** Trust Circle - Inner Circle  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Define a "core" circle whose messages get special treatment - no disappear timer, special archive, priority. Like your inner circle at home.

---

## PLAN

### Step 1: Designation
1. User can mark any Circle as "Trust Circle"
2. Max 1 per user
3. Clear warning: "This is special"

### Step 2: Special Treatment
- Messages NEVER auto-delete
- Separate archive (never cleaned)
- Priority notifications (ringtone)
- Show "core" indicator

### Database
```sql
ALTER TABLE circle_members ADD COLUMN is_trust_circle BOOLEAN DEFAULT FALSE;

ALTER TABLE messages ADD COLUMN is_core_message BOOLEAN DEFAULT FALSE;
```

### Step 3: UI
1. Mark Circle UI in settings
2. Special badge on Circle
3. "Core" filter in DMs

### Step 4: Migration
1. Existing messages can be migrated
2. Bulk toggle option
