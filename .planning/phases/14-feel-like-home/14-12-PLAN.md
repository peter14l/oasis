# 14-12: Warm Whisper

**Feature:** Warm Whisper - Care Ping  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Send a "warm note" - encrypted care ping that shows you care without requiring a response. Gentle notification: "Someone sent you warmth" - like someone thinking of you.

---

## PLAN

### Step 1: Core Mechanic
1. One-way: no reply required
2. Can add short text (max 100 chars)
3. Or just a "wave" / emoji
4. Anonymous option available

### Step 2: Delivery
1. Delivered as "Warm whisper received" notification
2. Tap to reveal (requires auth)
3. Can be saved or dismissed
4. No "seen" indicator

### Database
```sql
CREATE TABLE warm_whispers (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  sender_id UUID REFERENCES auth.users(id),
  recipient_id UUID REFERENCES auth.users(id),
  message TEXT,
  is_anonymous BOOLEAN DEFAULT FALSE,
  revealed_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### Step 3: Rate Limiting
1. Max 3 whispers per day
2. Cannot spam
3. Block/report option

### Step 4: UI
1. Long-press or special button in chat
2. Quick emoji picker
3. Anonymous toggle
4. Inbox for received whispers

### Step 5: Ideas for "Warm"
- 💛 "Thinking of you"
- 🌟 "You're amazing"
- 🍀 "Good vibes sent"
- 🤗 "Just wanted to say hi"
- 💪 "You've got this"
