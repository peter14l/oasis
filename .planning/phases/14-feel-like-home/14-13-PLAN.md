# 14-13: Check-in Pulse

**Feature:** Check-in Pulse - Location-Free Presence  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Status check-ins without GPS - "Home", "Work", "Traveling", "With [friend]". No location tracking, just presence sharing.

---

## PLAN

### Step 1: Status Types
1. 🏠 "Home"
2. 💼 "Work"
3. 🚗 "Traveling"
4. 👥 "With [friend]" - select from contacts
5. 🎯 "At [location]" - text only, no coordinates

### Step 2: How It Works
1. User selects status + optional text
2. Shows in profile: "🟢 Home"
3. Friends see on profile + messages
4. Auto-reset after X hours or manual

### Database
```sql
ALTER TABLE profiles ADD COLUMN pulse_status VARCHAR(50);
ALTER TABLE profiles ADD COLUMN pulse_text VARCHAR(100);
ALTER TABLE profiles ADD COLUMN pulse_since TIMESTAMPTZ;
```

### Step 3: Notifications to Friends
1. When friend comes online, show their pulse
2. Optional: nudge if "traveling" and you're free

### Step 4: UI
1. Quick picker in header
2. Profile shows current pulse
3. Toggle visibility

### Step 5: NO GPS
- Pure manual input
- Text is just text
- No location data collected
