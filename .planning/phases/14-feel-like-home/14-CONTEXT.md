# Feel Like Home - Personal Sanctuary Features

## Phase Overview

**Goal:** Create features that make Oasis feel like a warm, personalized home - a digital sanctuary where users belong.

**Phase:** 14  
**Status:** Planning in Progress  
**Plans:** 14 feature plans

## Features

### 1. Mood Orbit - Personal Vibe Widget
A personal widget on the feed showing the user's current vibe/mood. Others can see it before engaging.

- Personal state: calm, energetic, need coffee, creative, introspective, etc.
- Customizable mood options
- Privacy controls (who can see)

### 2. Safe Haven Badge
Trust badges earned for creating safety bubbles - showing you're a trusted member.

- Safety contributor badges
- Recognition for helping new members
- Trust score display

### 3. Digital Garden
Private space to "plant" thoughts that grow over time.

- Seed thoughts → grow into insights
- Private vs friend-visible views
- Growth visualization

### 4. Cozy Hours
Custom "do not disturb" statuses with personality.

- Options: "in mycocoon", "reading mode", "offline recharge", "movie night"
- Custom status messages
- Auto-reply suggestions

### 5. Circle Warmth
Each Circle shows a "temperature" based on interaction warmth.

- Warmth algorithm (participation, response time, mood matching)
- "Cold" suggestions to reach out
- Temperature visualization

### 6. Welcome Wagon
Customizable welcome templates for new connections.

- Privacy-first welcome message
- Custom templates
- Quick reply options

### 7. Memory Lane
Auto-collage "this time last year" from posts.

- Annual memory generation
- Share or keep private
- Multiple years support

### 8. Quiet Rooms
Sub-groups within Circles for different moods/topics.

- "Late night thoughts"
- "Happy news"
- "Advice needed"
- Topic-based rooms

### 9. Fortress Mode
One-tap lock with custom "away" message.

- PIN/password protection
- Custom away messages
- Friends see you're "in your fortress"

### 10. Trust Circle
Inner circle with special treatment.

- No disappear timer
- Special archive
- Priority from core

### 11. Digital Will
Legacy planning for digital presence.

- Designate legacy contact
- Auto-delete timeline
- Data transfer instructions

### 12. Warm Whisper
Care ping without requiring response.

- Gentle notification: "Someone sent you warmth"
- Encrypted note
- Anonymous option

### 13. Check-in Pulse
Location-free presence status.

- "Home", "Work", "Traveling", "With [friend]"
- No GPS tracking
- Social check-ins

### 14. Home Base
Customizable personal avatar space.

- Interactive space
- Friend visits allowed
- Customize items/themes
