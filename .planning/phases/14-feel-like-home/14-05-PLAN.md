# 14-05: Circle Warmth

**Feature:** Circle Warmth - Temperature Indicator  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Each Circle shows a "warmth" temperature based on interaction patterns. Cold = reach out, warm = active, hot = thriving. Helps users know where to invest energy.

---

## PLAN

### Warmth Algorithm
```
Warmth = (recent_posts * 0.3) + (messages * 0.3) + (mood_match * 0.2) + (recency * 0.2)
- Based on last 7 days activity
- Mood match = similar moods with other members
- Recency = days since last interaction
```

### Database
```sql
ALTER TABLE circles ADD COLUMN warmth_score DECIMAL(3,2) DEFAULT 0.5;
ALTER TABLE circles ADD COLUMN calculated_at TIMESTAMPTZ;
```

### Step 1: Warmth Calculation Function
1. Supabase function to calculate warmth
2. Run nightly for all circles
3. Cache result

### Step 2: UI
1. Show thermometer indicator on Circle card
2. Color gradient: blue (cold) → yellow (warm) → orange (hot)
3. "Cold" prompt: "This circle could use some warmth 💛"

### Step 3: Suggestions
1. When warmth is low, suggest:
   - "Start a conversation"
   - "Share how you're feeling"
   - "Ask how someone's doing"
