# 14-04: Cozy Hours - Custom DND

**Feature:** Cozy Hours - Custom Do Not Disturb  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Personalized "do not disturb" with warm, personality-filled statuses instead of cold "offline". Like closing curtains - cozy, not distant.

---

## PLAN

### Predefined Cozy Modes
1. 🏠 "In my cocoon" - Resting
2. 📖 "Reading mode" - Focused
3. 🔋 "Offline recharge" - Low energy
4. 🎬 "Movie night" - Entertainment
5. 💭 "Deep thought" - Introspective
6. 🌙 "Sleepy" - Going to sleep

### Step 1: Database
```sql
ALTER TABLE profiles ADD COLUMN cozy_status VARCHAR(50);
ALTER TABLE profiles ADD COLUMN cozy_status_text VARCHAR(200);
ALTER TABLE profiles ADD COLUMN cozy_until TIMESTAMPTZ;
```

### Step 2: Provider
1. Create `CozyModeProvider`
2. `setCozyMode(status, customText, duration)`
3. Auto-expire after time

### Step 3: UI
1. Quick toggle in header
2. Status appears on profile
3. DMs get auto-reply: "[Name] is in reading mode - they'll get back to you soon"

### Step 4: Notification Handling
1. Show "cozy" indicator to message senders
2. Delay non-urgent notifications
3. Clear indicator for urgent
