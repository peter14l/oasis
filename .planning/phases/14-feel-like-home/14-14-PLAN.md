# 14-14: Home Base

**Feature:** Home Base - Personal Avatar Space  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Each user has a customizable "digital home" - a personal visual space with interactive items that friends can visit. Like a personal website or virtual room.

---

## PLAN

### Step 1: The Base Concept
1. 2D isometric space
2. Furniture/items can be placed
3. Friends can visit and interact
4. Leave "signatures" on guestbook

### Step 2: Customization
Items to place:
- 🏠 Furniture: bed, couch, desk, plant, rug, lamp
- 🎨 Decorations: photos, art, achievements
- ⭐ Status symbols: badges, vibes
- 🚪 Portal to other bases

### Database
```sql
CREATE TABLE home_bases (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  layout JSONB,  -- item positions
  theme VARCHAR(50) DEFAULT 'default',
  visitors_count INTEGER DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE home_items (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  item_type VARCHAR(50),
  emoji VARCHAR(10),
  is_premium BOOLEAN DEFAULT FALSE
);
```

### Step 3: Visits
1. "Visit [name]'s home" button on profile
2. Guestbook to sign
3. Can leave reactions
4. Host gets notification

### Step 4: Progression
1. Unlock more items over time
2. Pro members get more items
3. Seasonal items (holidays)

### Step 5: Simplified Version (MVP)
- Just background + small avatar
- Guestbook only
- Profile link "Visit my home"
