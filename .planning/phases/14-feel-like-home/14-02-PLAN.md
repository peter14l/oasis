# 14-02: Safe Haven Badge

**Feature:** Safe Haven Badge - Trust Badges  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Create a badge system that recognizes members who create safety and support in Oasis. Shows users are trusted community members.

---

## CONTEXT

### Existing
- Existing trust/security features in `lib/features/auth/`
- Community service: `lib/services/community_service.dart`

### Database
```sql
CREATE TABLE trust_badges (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name VARCHAR(50) NOT NULL,
  description TEXT,
  icon VARCHAR(10),  -- emoji
  criteria JSONB,  -- criteria for earning
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE user_badges (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  badge_id UUID REFERENCES trust_badges(id),
  earned_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(user_id, badge_id)
);
```

---

## PLAN

### Step 1: Badge Definitions
1. Define badge types:
   - 🌿 **Safe Space** - Created a trusted Circle
   - 🛡️ **Shield** - Reported harmful content (and it was verified)
   - 🌟 **Welcomer** - Helped 10+ new members
   - 💚 **Calm Creator** - No violations in 6 months
   - 🔒 **Privacy Guard** - Enabled all privacy features

### Step 2: Database
1. Create badges table
2. Create user_badges junction table
3. RLS: anyone can read, only system can write

### Step 3: Badge Award Logic
1. Create Supabase function `check_badge_eligibility(user_id)`
2. Run periodically (daily) or on trigger events
3. Award badges automatically

### Step 4: UI Display
1. Create BadgeWidget component
2. Show on profile (badges section)
3. Show in Circles (member list badges)
