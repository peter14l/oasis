# 14-08: Quiet Rooms

**Feature:** Quiet Rooms - Sub-groups within Circles  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Create topic/mood-based "rooms" within Circles. Users can join rooms that match their current vibe - "Late night thoughts", "happy news", "advice needed".

---

## PLAN

### Room Types
1. 💭 "Late Night Thoughts" - Deep conversations
2. 🎉 "Happy News" - Share wins
3. 🙋 "Advice Needed" - Ask the community
4. 📚 "Book Club" - Reading discussions
5. 🎵 "Music Share" - Share tracks
6. 💪 "Motivation" - Encouragement

### Database
```sql
CREATE TABLE circle_rooms (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  circle_id UUID REFERENCES circles(id),
  name VARCHAR(50) NOT NULL,
  description TEXT,
  emoji VARCHAR(10),
  is_active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE room_members (
  room_id UUID REFERENCES circle_rooms(id),
  user_id UUID REFERENCES auth.users(id),
  joined_at TIMESTAMPTZ DEFAULT NOW(),
  PRIMARY KEY(room_id, user_id)
);
```

### Step 1: Room Management
1. Circle admins can create rooms
2. Members join rooms they want
3. Posts can be tagged to rooms

### Step 2: UI
1. New tab in Circle: "Rooms"
2. Room list with members count
3. Post composer with room selector

### Step 3: Moderation
1. Room-specific rules
2. Mute room option
3. Room notifications
