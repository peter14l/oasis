# 14-07: Memory Lane

**Feature:** Memory Lane - This Time Last Year  
**Phase:** 14 - Feel Like Home  
**Status:** Planning  

---

## TASK

Auto-generate a collage "this time last year" from user's posts. Share the memory or keep private. Creates nostalgia and reflection.

---

## PLAN

### Step 1: Post Aggregation
1. Query posts from 1 year ago (± 7 days)
2. Group by date/theme
3. Generate collage image (canvas)

### Step 2: Generation
1. Daily job runs for each user with posts from year ago
2. Creates "Memory Lane" card
3. Shows on anniversary date

### Database
```sql
CREATE TABLE memories (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  post_ids UUID[],  -- posts included
  generated_at TIMESTAMPTZ DEFAULT NOW(),
  was_shared BOOLEAN DEFAULT FALSE
);
```

### Step 3: UI
1. On anniversary day, show Memory Lane at feed top
2. "On this day..." header
3. Share button (share to story)
4. Dismiss button

### Step 4: Multi-Year
1. Support 1yr, 2yr, 3yr
2. "View more memories" option
3. Best memories highlight
