4. **Success step:**
   - Show success message
   - Button to proceed to app (or prompt to set up new recovery key)

**Flow:** 
1. User enters email + password → Verify with Supabase Auth
2. If valid → Show warning screen with checkbox
3. If confirmed → Show new PIN entry screen
4. User enters new PIN twice → Validate → Call VaultService.resetPin()
5. Show success → Optionally prompt to set new recovery key via EncryptionSetupScreen

Use Material 3 design, match existing auth screen styling.
