4. **Processing step:**
   - Show loading while generating new keys
   - Call `EncryptionService().generateNewKeysWithPin(newPin)`

5. **Success step:**
   - Show success message
   - Show NEW recovery key via RecoveryKeySheet
   - Button to proceed to app

**Flow:** 
1. User enters email + password → Verify with Supabase Auth
2. If valid → Show warning screen with checkbox
3. If confirmed → Show new PIN entry screen
4. User enters new PIN twice → Validate → Call EncryptionService.generateNewKeysWithPin()
5. Show success → Show NEW recovery key → Navigate to app
